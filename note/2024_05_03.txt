 HTML
- HyperTextMarkupLanguage의 약자
- 웹페이지를 만드는 대표적인 마크업언어
- 웹페이지의 구조를 표현(트리구조)

HTML의 요소
- 태그로 이루어져 있음 
HTML 요소는 일반적으로 시작 태그와 닫는 태그로 정의됨

HTML의 주석
<!--이게 주석이다-->
- 브라우저에서 출력이 되지 않는 문장
--- 소스 코드에 대한 설명을 할 때 사용

<head> <!-- HTML 페이지의 meta 정보(페이지에 대한 정보)를 포함, 초기 페이지 렌더링시에 불러와야 할 외부 링크 정의 -->
<meta charset="UTF-8"> - 한글 인코딩 방식 
width=device-width - 기기에 따라 맞춤
속성은 항상 시작 태그에 입력

HTML의 속성(attributes)
- 모든 HTML 요소들은 속성을 가지고 잇음
- 속성은 HTML 요소에 대한 추가적인 정보를 제공
- 특정 태그에서의 속성이나 사전 정의된 속성은 적용된 기능으로서 추가정보를 활용

ex)
<img src="경로"></img>

직접 속성을 만들 수 있음
ex) <div data-sido="인천광역시"> </div>

HTML 속성 권장사항
- 소문자로 사용
- 속성은 소문자, 대문자 상관없이 인식하나 소문자로 쓰는 것을 권장

HTML Block & Inline 요소

Block 요소는
- 닫는 태그 후 개행
- 공간을 지정할 수 있음(너비와 높이를 가질 수 있음)
- 아래 위 또는 왼쪽 오른쪽에 공백 지정

Inline 요소
- 줄 개행 x
- 공간을 지정할 수 없음
- 요소 안에 있는 내용만큼의 공간만 차지
- 위 아래 공백을 지정할 수 없으나 내부 공백은 지정 가능 줄 개행 x
