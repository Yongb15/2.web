HTML -> 뼈대 작성

CSS
- HTML이나 XML 같은 구조화된 문서를 어떻게 렌더링 할 것인지 정의하기 위한 언어
- CSS는 HTML의 각 요소의 스타일을 정의하여 화면 등에 어떻게 렌더링하면 되는지 브라우저에게 설명하기 위한 언어
- CSS(Cascading Style Sheet)

HTML에서 CSS를 적용하는 방법
- HTML과 CSS는 각자의 문법을 갖는 별개의 언어이며 HTML CSS를 포함할 수 있다!
- 그러나 HTML없이 단독으로 존재하는 CSS는 의미가 없음!

CSS 기본 문법
- 선택자(Selector)
    ㄴ CSS를 적용하고자 하는 HTML 요소
- 선언부
    ㄴ 하나 이상의 선언들을 세미콜론(;)으로 구분하여 포함할 수 있으며, 중괄호({})를 사용하여 전체를 둘러쌈
    ㄴ 각 선언은 CSS 속성명(Property)과 속성 값(value)을 가지며 그 둘은 콜론(:)으로 연결

셀렉터(Selector) { 선언 블록(Declaration block)    
    ex) color: red; 선언(Declaration)
        color : 프로퍼티(Property)
        red;  : 값(Property value)

        선언은 여러 개 가능!
}

CSS 선언 방식
- 인라인(In-line) 방식
    ㄴ HTML의 각 태그에서 Style 속성에 직접 작성하는 방식0
- 내장(Embedded) 방식
    ㄴ 같은 HTML 문서에 <style> ~ </style> 태그 안에 스타일을 적용하는 방식
- 링크(Link) 방식
    ㄴ HTML을 이용하여 외부 문서로 CSS를 불러와 적용하는 방식

@import 방식
- @import를 이용하여 외부 문서로 CSS를 불러와 적용하는 방식

CSS 선택자
- 요소 선택자
    ㄴ CSS를 적용할 대상으로 HTML 요소의 이름을 직접 선택하여 사용할 수 있다!
    ㄴ ex) <h2> 이 부분에 스타일을 적용한다.</h2>
            h2 { color : red; }
- 클래스 선택자
    ㄴ 특정 집단의 여러 요소를 한 번에 선택할 때 사용
    ㄴ 이런 특정 집단을 묶을 수 있는 걸 클래스(class)라고 하며 같은 클래스 이름을 가진 요소들을 모두 선택해줌
    ㄴ .을 써서 구분
id
- CSS를 적용할 대상으로 특정 요소를 선택할 때 사용
- #을 써서 구분
주의
- HTML과 CSS에서는 하나의 웹페이지 속하는 여러 요소에 같은 아이디 이름을 사용해도 문제X
- 하지만 중복된 아이디를 이용해 자바스크립트 작업을 하게 되면 오류 발생
- 되도록 하나의 웹 페이지에 속하는 요소에는 다른 아이디 이름을 사용하거나 클래스를 사용하는게 좋다

그룹(Group) 선택자
- 위에서 언급한 여러 선택자를 같이 사용할 때 사용
- 여러 선택자를 쉼표(,)로 구분하여 연결
- 그룹 선택자는 코드를 중복해서 작성하지 않도록 코드를 간결하게 만들어줌

전체 선택자
- '*' 문자를 사용하여 HTML 문서 내의 모든 요소를 선택

스타일 적용 우선순위
- 인라인 선언 방식
- 아이디 선택자
- 클래스 선택자
- 태그 선택자
- 전체 선택자
인라인 > 아이디 > 클래스 > 태그 > 전체

테두리(border)

폰트(font)

★ 내부 여백(padding)
- 내용과 테두리 사이의 간격의 크기를 설정해주는 속성
- padding을 사용하다보면 요소의 크기가 생각보다 커질 수 있음

★ 외부 여백(margin)
- 요소와 주변 여백
- margin 영역이 겹쳤을 때 크기가 큰 margin이 먹음
- 안드로이드에서는 둘의 합한 margin 적용
- margin을 사용하다보면 요소의 크기가 생각보다 작아질 수 있음
